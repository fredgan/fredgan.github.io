优质软件列表

经过多年实践，总结的优质软件列表。持续更新中…

一、Windows

1.1 Directory Opus（收费）

功能超强的资源管理器。比Windows自带的资源管理器好用太多，多Tab，正则表达式批量重命名，FTP连接器，多窗口操作，图片文本文件预览等。

1.2 SQLyog

连接MySQL数据库的客户端软件。软件小巧，功能强大，界面清晰。

1.3 Mind Master

思维导图工具。最好用的思维导图工具，国产骄傲。同时具有Mac 移动端版本。

1.4 snipaste

剪贴板增强工具。F1快速截图，截图工具功能强大，F3贴图到桌面。可用于内容对比，或者复杂截图。

1.5 XShell

SSH远程工具。虽然多年前暴露曾经有后台上传隐私数据问题，但就软件体验而言，确实是一款体验超好的SSH工具

1.6 Sublime Text

简洁的并且功能强大的编辑器，支持强大的正则表达式和各种插件。

二、Mac

2.1 Mind Master

参考 Windows Mind Master

2.2 Sequal Pro

Mac上好用的MySQL数据库连接客户端。

2.3 Maipo

好用的第三方微博客户端

2.4 VLC

视频播放器

2.5 IINA

开源视频播放器，几乎可以支持所有格式的视频播放

2.6 calibre

开源软件，最好用的 mobi/epub 电子书阅读器。也支持azw3格式和pdf格式。 另外也是一个优秀的电子书管理软件，自定义标签，批量标注。可以打造一个自己的专属电子图书馆。

PS：电子书可以到 鸠摩搜索 下载，一个强大的搜索下载网站。

2.7 Eudic

欧路词典，最好用的词典和单词学习软件。
